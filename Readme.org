* Dotfiles

My personal dotfiles  when I setip a new environnement (Ubuntu)

Tested on :
- Ubuntu 20 LTS
- Ubuntu 22 LTS





** Pre Installl

At first install =Git=

#+begin_src shell
  sudo add-apt-repository -y ppa:git-core/ppa
  sudo apt-get update
  sudo apt-get install -y git-all git-extras
#+end_src
** Installation

*** Clone git repository

#+begin_src shell
git clone https://github.com/nchapon/dotfiles.git dotfiles
#+end_src

*** Install essentials

#+begin_src shell
  cd dotfiles
  ./ubuntu-setup.sh 
#+end_src

*** Install OhMyZsh

See instructions here : https://ohmyz.sh/#install

*** Link my configuration files

#+begin_src shell
  cd ~/dotfiles
  stow git
  stow emacs
  stow zsh
#+end_src
