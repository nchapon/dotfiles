# dotfiles

Personal dotfiles used when I setup a new environnement.

## Usage

> git clone https://github.com/nchapon/dotfiles.git .dotfiles

> cd .dotfiles/

> ./setup.sh

## Local Settings ##

Stash your environment variables and others sepcific settings in `~/.bash.local`
